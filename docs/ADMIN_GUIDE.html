Guide Administrateur • Padel Paradise

Version 1.0

Accès administrateur

URL : club.padelparadise.local (ou /club depuis votre domaine).
Identifiant par défaut : admin@padelparadise.club
Mot de passe par défaut : ClubPadel!2025
Pensez à modifier ce mot de passe lors de votre première connexion via l'équipe technique.

1. Réservations & planning

  • Filtrez par date grâce au sélecteur situé en haut de l'écran « Planning ».
  • Les réservations en conflit sont automatiquement bloquées par le système.
  • Le paiement partagé affiche le montant dû par chaque participant.

2. Gestion des membres

  • Consultez le niveau, le nombre de matchs joués et les points internes de chaque membre.
  • Les nouveaux inscrits sont signalés dans le tableau de bord.
  • Utilisez la fonctionnalité « Communauté » du site public pour envoyer des messages ciblés.

3. Suivi financier

  • Toutes les transactions SumUp sont listées dans l'onglet « Finances ».
  • Le statut pending_split indique un paiement partagé en attente de complétion.
  • Chaque transaction inclut une référence (ex. SUMUP-XXXXXX) permettant un rapprochement comptable.

4. Matchs & communauté

  • Les organisateurs peuvent publier les résultats directement depuis leur espace joueur.
  • Les statistiques (victoires, temps de jeu) se mettent à jour automatiquement.
  • Utilisez les notifications personnalisées pour informer vos joueurs (annulations, nouveaux tournois...).

5. Support & formation

Une vidéo de prise en main est accessible dans l'onglet « Ressources ». Pour toute question, contactez l'équipe digitale Padel Paradise via support@padelparadise.club.
